Postup instalace

Projekt byl vytvořen za pomocí frameworku Laravel a Vue.js

Jako vývojové prostředí byl zvolen Laravel homestead, kdy IS běží na virtuálním stroji, který je připraven pomocí vagrant boxu.
Zprovoznění Laravel homestead je nejlépe vysvětleno v následujícím odkazu. Po instalaci a založení projektu stačí pouze vložit odevzdané soubory do projektové složky a použít příkazy pro stažení knihoven a vytvoření iniciálního stavu databáze.

npm install - Stažení node_modules

php artisan migrate:fresh - vytvoření databázové struktury

php artisan db:seed - vložení ukázkových dat do databáze
